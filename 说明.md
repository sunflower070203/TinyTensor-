# 项目说明书 — 极简 C++ 自动求导系统

## 一、这个项目是什么

这是一个用纯 C++17 实现的教学级自动求导系统，不需要任何第三方库。

它能做什么？三件事：

1. **张量运算**：支持标量、向量、矩阵的加减乘除、矩阵乘法、转置、求和等操作，类似一个简化版的 PyTorch。
2. **自动求导**：前向计算时自动构建计算图，调用 `backward()` 时通过拓扑排序 + 链式法则反向传播梯度。这是深度学习框架的核心。
3. **神经网络训练 + 可视化**：内置 Linear 层、SGD 优化器、MSE 损失函数，可以训练 XOR 和同心圆分类任务，并生成决策边界的可视化图片或实时动画。

整个库的核心代码只有一个文件：`Tensor/tensor.hpp`，约 400 行。

## 三、每个文件具体干什么

### Tensor/tensor.hpp — 自动求导核心库

这是整个项目最重要的文件，约 400 行代码，实现了：

| 组件 | 内容 | 代码行数 |
|------|------|----------|
| `struct Node` | 计算图节点：数据、梯度、父节点、反向函数 | ~6 行 |
| `class Tensor` | 张量类：构造、访问、运算符重载 | ~200 行 |
| `operator+` `-` `*` `/` | 标量/矩阵运算，自动构建计算图 | 各 ~20 行 |
| `matmul()` | 矩阵乘法 + 反向传播 | ~25 行 |
| `relu()` `sigmoid()` `tanh()` `softmax()` | 激活函数 | 各 ~5 行 |
| `backward()` | 拓扑排序 + 逆序执行反向函数 | ~10 行 |
| `class Linear` | 全连接层：x @ W + b，Kaiming 初始化 | ~15 行 |
| `class SGD` | 随机梯度下降优化器 | ~15 行 |
| `mse_loss()` | 均方误差损失函数 | 1 行 |

**工作原理**：每次运算（如 `a + b`）会创建一个新的 Tensor 节点，记录父节点指针和一个 lambda 闭包（局部反向传播函数）。调用 `loss.backward()` 时，从 loss 节点做拓扑排序，逆序执行每个节点的闭包，把梯度累加到父节点中。

### Tensor/main.cpp — 训练示例

三个训练任务，展示从简单到复杂的完整能力：

1. **线性回归**（`train_linear_regression`）：用标量 w、b 拟合 y = 2x + 1，200 轮
2. **XOR 分类**（`train_xor`）：两层网络 `Linear(2,8) → tanh → Linear(8,1) → sigmoid`，2500 轮
3. **同心圆分类**（`train_circle_classifier`）：两层网络，1800 轮，准确率 > 90%

### Tensor/tests.cpp — 测试文件

6 个测试函数，用 `assert()` 验证：

| 测试 | 验证内容 |
|------|----------|
| `test_scalar_autograd` | 标量链式求导：z = (xy + x³)/y 的梯度 |
| `test_matrix_broadcast_and_matmul` | 矩阵乘法 + 广播梯度 |
| `test_activations_and_softmax` | relu、sigmoid、tanh、softmax 输出正确性 |
| `test_linear_regression_converges` | 线性回归收敛到 w≈2, b≈1 |
| `test_xor_training` | XOR 预测 < 0.25 或 > 0.75 |
| `test_circle_training` | 同心圆准确率 ≥ 90% |

### Tensor/interactive.cpp — 交互式终端

命令行 REPL，支持手动创建变量、执行运算、反向传播、查看梯度、训练模型。用于课堂演示或调试。

### visualization/image_writer.hpp — 图片写入工具

两个函数：
- `write_ppm()`：写 PPM 格式（最简单的图片格式）
- `write_bmp()`：写 BMP 格式（Windows 双击可打开）

### visualization/plot.hpp — 绘图工具

三个函数：
- `plot_decision_boundary()`：生成决策边界热力图（蓝色=0，红色=1）
- `plot_classification()`：热力图 + 训练数据散点叠加
- `plot_loss_curve()`：Loss 折线图

### visualization/live_window.hpp — 实时窗口

封装 Win32 GDI API，创建一个 800×600 的窗口，支持：
- 逐像素绘制热力图
- 画圆、画线、画矩形
- GDI 文字渲染
- 实时刷新 + 消息泵（保持窗口响应）

### visualization/train_and_export.cpp — 静态导出程序

运行训练 → 生成 BMP 图片文件。运行后 `output/` 目录下出现 4 张图片。

### visualization/train_live.cpp — 实时训练程序

弹出 Win32 窗口，每 50 个 epoch 刷新一次决策边界，实时看到训练过程。

## 四、如何运行

### 环境要求

- Windows 10/11
- MinGW-w64 (g++ 支持 C++17)

### 编译命令

```powershell
cd C:\Users\Sunny\.vscode\C++\my_clean_repo

# 基础版：测试
g++ -std=c++17 Tensor/tests.cpp -O2 -o tensor_tests.exe

# 基础版：训练示例
g++ -std=c++17 Tensor/main.cpp -O2 -o tensor_demo.exe

# 基础版：交互式终端
g++ -std=c++17 Tensor/interactive.cpp -O2 -o tensor_cli.exe

# 挑战版：静态图片导出
g++ -std=c++17 -I Tensor -I visualization visualization/train_and_export.cpp -O2 -o train_export.exe

# 挑战版：实时训练窗口
g++ -std=c++17 -I Tensor -I visualization visualization/train_live.cpp -O2 -lgdi32 -o train_live.exe
```

### 运行命令

```powershell
.\tensor_tests.exe          # 运行测试，看到 "All tests passed"
.\tensor_demo.exe           # 运行训练示例，输出预测结果和准确率
.\train_export.exe          # 生成 BMP 图片到 output/ 目录
.\train_live.exe            # 弹出实时训练窗口
```

### 查看结果

```powershell
mspaint output\xor_boundary.bmp       # 用画图打开 XOR 决策边界
mspaint output\circle_boundary.bmp    # 用画图打开同心圆决策边界
```

## 五、技术要点

### 自动求导原理

1. 前向传播：每个运算创建新节点，记录父节点和局部反向函数
2. 反向传播：从 loss 节点拓扑排序，逆序执行局部反向函数
3. 梯度累加：`+=` 操作，正确处理 DAG 中一个节点被多次使用的情况
4. 广播处理：标量 bias 加到 batch 输出时，梯度按维度折叠求和

### 决策边界生成原理

1. 在 [-1.5, 1.5] × [-1.5, 1.5] 区域生成 200×200 网格
2. 对每个网格点做一次前向预测，得到 [0, 1] 的概率值
3. 将概率映射为颜色：0 → 蓝色，0.5 → 白色，1 → 红色
4. 逐像素写入 BMP 文件

### 实时窗口原理

1. 创建 Win32 窗口 + DIB 像素缓冲
2. 训练循环中每 50 epoch 刷新一次决策边界
3. 每 5 个 epoch 调用 `yield()`（Sleep + 处理窗口消息），防止界面卡死
4. 关闭窗口时训练自动停止
